Intel Lookup Demo

Browser-side form, server-side keys. The form below POSTs to /api/intel/lookup, a Cloudflare Pages Function that adds the provider API key from a Worker secret and forwards to the upstream service. The browser never sees the key.

What this is. A live, rate-limited demo (10 lookups/min/IP by default). Currently wired: AbuseIPDB for IP reputation and ip-api.com for geo/ASN enrichment (no key required). The full PowerShell stack on the walkthrough page covers more providers and runs against your own keys when cloned.
