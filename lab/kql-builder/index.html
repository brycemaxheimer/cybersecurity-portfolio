KQL Builder

Coming soon.

Visual KQL composition tool — pick a table, an Event ID, a time range, the columns you want back, and any filters; the page emits the equivalent KQL and lets you copy it (or send it straight to the playground).

What it'll cover

  • Source table (DeviceLogonEvents, DeviceProcessEvents, SecurityEvent, SigninLogs, etc.)
  • Time range (last 1h / 24h / 7d / custom)
  • EventID and ActionType picker (when the source table has those)
  • Filter rows: column + operator + value, multiple filters AND-ed together
  • Output columns selector (all / pick a subset)
  • Toggle between detailed and summarized output
  • Save and load named queries to local browser storage
  • "Send to playground" button that opens the KQL playground with the query pre-populated

Origin

This is a port of my KqlBuilder.ps1 WinForms module. The PowerShell version uses dropdowns, the same logic; this version replaces the WinForms layer with HTML form controls and stores queries in localStorage instead of MS Access.

Status: not yet built. Underlying source: SOC Dashboard / Modules / KqlBuilder.ps1.
